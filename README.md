# vcs

This package is based on version control systems.

## Usage

- help

```bash
npm start help
```

- init

```bash
npm start init
```

> creates `.vcs` and `.vcs/obj/`

- add

```bash
npm start add path/to/file
```

> creates a hash and compressed file and maps it in `.vcs/index`
